### Test environment:
python 3.8
ubuntu 20.10

### Installation


1. Create environment (Optional/Recommended)
   ```sh
   python3.8 -m venv venv #create
   source venv/bin/activate #use
   ```

2. Install libraries
   ```sh
   python -m pip install -r py_freeze_req.txt
   ```

3. Run
   ```sh
   python candidates_viz_api.py #Run the flask on 8080 port
   ```
